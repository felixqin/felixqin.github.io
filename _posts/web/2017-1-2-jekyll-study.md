# 安装命令

## 安装

`
sudo pacman -S ruby
gem install jekyll
gem install bundler
gem install jekyll-sitemap
gem install pygments.rb
`

## 配置
~/.bashrc PATH 加入：
`
$(ruby -e 'print Gem.user_dir')/bi
`

## 运行
`
jekyll --version
jekyll serve --host=0.0.0.0
`
